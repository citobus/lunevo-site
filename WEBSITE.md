# lunevo Website Notes

## Current scope

This folder contains the first static marketing site for lunevo.

- `index.html` is the landing page
- `styles.css` holds the visual system and responsive layout
- `script.js` adds lightweight reveal animations

## Design direction

- Keep the site calm, airy, and warm
- Mirror the app's sunrise-to-night gradient language
- Prefer simple sections with generous whitespace
- Avoid heavy product-marketing language

## Good next additions

- Add real app screenshots or device mockups
- Create a more detailed features section
- Add a waitlist or contact form
- Add an about/story section for the product vision
- Add deployment notes once hosting is chosen

## Content guardrails

- Write like a thoughtful guide, not a hype-driven startup page
- Keep copy short and easy to scan
- Focus on rhythm, energy, flow, and well-being
- Stay visually simple even as more sections are added

## Deployment ideas

- GitHub Pages
- Netlify
- Vercel

## Notes

If this grows into a larger site, we can convert it into a component-based setup later without losing the visual direction established here.
